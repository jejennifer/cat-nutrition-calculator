# 🐾 貓咪每日營養計算機

這是一個以 **Streamlit** 製作的互動式貓咪營養分析與食材建議工具，  
可幫助飼主根據貓咪體重與飲食型態自動計算每日營養需求，並推算鮮食與乾糧的建議餵食克數。

---

## ✨ 功能特色

- 🧮 **自動計算每日 RER / MER 與營養需求**
- 🥣 **乾糧熱量扣除與營養對照**
- 🥗 **鮮食自動配方建議（依 AAFCO 比例：65% 蛋白、22.5% 脂肪、12.5% 碳水）**
- 📊 **自動整合多食材營養組成與熱量分配**
- 📁 **乾糧與鮮食資料庫獨立管理**
- 🔍 **支援單位自動換算與 Atwater 估算**

---

## 🧠 計算邏輯摘要

- **RER（Resting Energy Requirement）= 70 × 體重^0.75**
- **MER 依年齡、結紮狀態與活動量調整係數**
- **鮮食配方以目標營養比例（65:22.5:12.5 kcal 基準）計算建議克數**
- **系統會自動整合乾糧提供的營養，推算剩餘鮮食補足量**
